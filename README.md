# QREye

QREye is a simple QR code generator built with Next.js.

## What it does

- Enter text or a URL
- Add more input fields if needed
- Generate a QR code
- Download the QR code as SVG, PNG, or JPG
- Switch between light and dark mode

## Run the project

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build for production

```bash
npm run build
npm start
```
